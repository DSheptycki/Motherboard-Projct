Components

Main Components

The Major elements to a motherboard are the CPU, RAM, expansion slots, heat sink and fan assembly, basic input/output system (BIOS) chip, chipset, and the circuitry that interconnects the motherboard components.

Connecting the Components

Conductive Wires or traces are used to connect the electronic components to a motherboard.

Bios Chip

The basic input/output system (BIOS) chip holds the program to a computers microprocessor that it uses to start the computer system after it is powered on. It also manages data flow between computers operating system (OS) and attached devices, such as the hard disk, video adapter, keyboard or mouse.
