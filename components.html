Components

Main Components

The Major elements to a motherboard are the CPU, RAM, expansion slots, heat sink and fan assembly, basic input/output system (BIOS) chip, chipset, and the circuitry that interconnects the motherboard components.

BIOS Chip

The basic input/output system (BIOS) chip contains the program to a computers CPU that it uses to start the computer system after it is powered on. It also manages data flow between the computers operating system (OS) and attached devices, such as the hard disk, video adapter, keyboard or mouse.

Expansion Slot

An expansion slot is a socket on the motherboard that is used to insert an expansion card (or circuit board), which provides additional features to a computer such as video, sound, advanced graphics, Ethernet or memory.

Connecting the Components

Conductive Wires or traces are used to connect the electronic components to a motherboard. These wires act as tracks for electricity to flow through between components on a motherboard. They are the paths for data to travel on the motherboard. These are the lines that you see on a motherboard.
