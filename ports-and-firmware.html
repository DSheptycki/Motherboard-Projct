Ports And Firmware

Firmware

Firmware is software that provides basic machine instructions that allows the hardware to function and communicate with other software running on a device. Firmware provides low-level control for a device's hardware.

Ports

The ports used on the case connect to the internal ports of the motherboard, also if a memory reader is installed it uses one of those ports for the ports on the case.

USB In Depth

the original USB 1.1 has a data transfer speed of 12MB per second, secondly USB 2.0 was created with a transfer speed of 480MB per second, and finally the current USB 3.0 uses 5 - 10GB of transfer speed per second.

SATA Technology

Sata technology is used to connect large storage devices into USB slots such as external hard drives or optical drives.
